Instalar n.Call no dispositivo

install_mobile

Atalho em modo app para o time de enfermagem

No Android e em desktop Chromium, o navegador pode abrir o prompt nativo de instalacao. No iPhone e iPad, a instalacao continua manual pelo menu Compartilhar.

Instalacao pendente Alertas inativos
download
Instalacao Use a URL publicada do GitHub Pages para que o navegador reconheca o prototipo como PWA.
notifications_active
Notificacoes Depois da instalacao, libere os alertas para receber banners nativos de nova chamada.
volume_up
Som no app O prototipo toca um alerta sintetizado quando a simulacao acontece com o app aberto.
vibration
Vibracao O app chama a API de vibracao. Android costuma suportar; iOS Safari nao expoe essa API.
Como instalar no iPhone ou iPad
  1. Abra a pagina pelo Safari.
  2. Toque em Compartilhar.
  3. Escolha "Adicionar a Tela de Inicio".

Limite do prototipo: som customizado so e garantido com o app aberto. Com o app fechado, a exibicao real de notificacoes depende de Push API e de um backend, por exemplo via FCM.

Validacao do prototipo As acoes de alerta e simulacao ficam aqui para nao poluir a tela principal.
